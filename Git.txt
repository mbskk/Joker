Git branches :
===========
git branch  => to list branches
git branch -a  => to list all branches including remote
git branch (branch name)  => to create new branch
git branch -d (branch name )  => to delete a branch
git branch -m (target branch name)  => to merge branches
git checkout (branch name)  => to switcg from branch to branch
git checkout -  => to switch last branch
git merge (active branch name)  => to merge in to a active branch
git merge (source branch) (target branch)  => to merge from source file to target file
git checkout - - (filename)  => to discard changes to a file
